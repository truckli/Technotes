\section{wget下载网站}
例如:
\begin{lstlisting}
wget -r -p -np -k http://www.21cn.com
wget --ftp-user=... --ftp-password=PASS  -r -p -np -k ftp://www.21cn.com
wget -r -p -np -k http://192.168.1.199/svn/OCTEON-SDK/branches/2012-03-27/OCTEON-SDK/docs/ --user=limz --password=123456
\end{lstlisting}
